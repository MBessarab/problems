# Заголовок столбца на листе Excel

Для заданного целого числа `columnNumber` верните название соответствующего столбца в том виде, в котором оно отображается на листе Excel.

Например:

```
A -> 1
B -> 2
C -> 3
...
Z -> 26
AA -> 27
AB -> 28
...
```

**Пример 1:**
<br/> Input: 1
<br/> Output: «A»

**Пример 2:**
<br/>Input: 28
<br/>Output: «AB»

**Пример 3:**
<br/>Input: 701
<br/>Output: «ZY»
